python网络爬虫（一）

requests库

1.安装

管理员权限cmd，运行

import requests
r=requests.get(“http://www.baidu.com“)
print(r.status_code) //状态码，返回值为200说明成功
r.encoding=’utf-8’ //改为utf8编码
print(r.text) //打印网页内容

hd={‘user-agent’:’Chrome/10’} Chrome浏览器第10个版本，模拟该浏览器向服务器发请求
r=requests.request(‘POST’,’http://python123.io/ws',heads=hd)

pxs={‘http’:’http://user:pass@10.10.10.1:1234‘
‘https’:’https://10.10.10.1:4321'}
r=requests.request(‘GET’,’http://www.baidu.com',proxies=pxs)
```
allow_redirexts：true/false，默认true，重定向开关
stream：true/false，默认true，获取内容立即下载开关
verify：true/false，默认true，认证SSL证书开关
cert：本地SSL证书路径

head方法

requests.head(url,**kwargs)

post方法

requests.post(url,data=None,json=None,**kwargs)

put方法

requests.put(url,data=None,**kwargs)

patch方法

requests.patch(url,data=None,**kwargs)

delete方法

requests.delete(url,**kwargs)

requests异常
r.raise_for_status() 如果不是200，产生异常requests.HTTPError
