字符串

字符串是单个字符的字符串序列，其他类型的序列还包括列表和元组。

序列的操作

  • 索引操作得到其各个元素 第一项索引为0，第二项为1，以此类推。

所以，以下两个操作是等效的

注意，可以在方括号中使用任意表达式，而不仅仅使用数字变量。

  • 分片
    一般形式为X[I:J]，表示“取出在X中从偏移量为I，直到但不包括偏移量为J的内容”，结果为返回一个新对象。在一个分片中，左边界默认为0，右边界默认为分片序列的长度。

注意：负偏移量如何用作分片的边界

  • 加号乘号
    作为一个序列，字符串支持加号进行合并（将两个字符串合成为一个新的字符串）。乘号是将原字符串重复多少次。
  • 不可变性
    字符串在python中具有不可变性——在创建后不能就地改变。例如：不能通过对其某一位置进行赋值而改变字符串，但是可以通过建立一个新的字符串并以同一个变量名对其进行赋值。python在运行过程中会清理旧的对象。

注意：以上操作均为一个真正的序列操作，同理，这些操作不局限于字符串，在python中其他序列中也会工作，包括列表和元组。以下，是字符串独有的一些操作方法。

字符串特有操作

  • find方法
    字符串find方法是一个基本的子字符串查找的操作（没有，返回-1。否则，返回一个传入子字符串的偏移量）
  • replace方法
    对全局进行搜索和替换，例如S.replace('pa','XYZ')将S中的pa全部替换为XYZ。S没有被改变
