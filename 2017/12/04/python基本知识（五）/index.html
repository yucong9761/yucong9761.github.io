Python数字类型

数字常量

允许整数具有无穷精度。十六进制以0x或0X开头，数字中字母大小写均可。八进制以0o或0O开头。二进制以0b或0B开头。内置函数hex(I)、oct(I)、bin(I)把一个整数转化为这三种进制表示的字符串。int(str,base)根据每个给定的进制把一个运行时字符串转换为一个整数。eval()把字符串转为十进制。

复数，python的复数常量写成实部+虚部，虚部以j或J结尾。可以通过complex(real,imag)来创建。

Python表达式操作符

  • 比较运算符可以连续使用：X<Y<Z等同于X<Y and Y<Z
  • X//Y会把余数的小数部分去掉，X/Y真正的除法。python3.0中若想得整数，用//，整除情况下/得x.0。//是向下取整，eg：2.4=2 -2.1=-3
  • python数字类型复杂度：整数比浮点数简单，浮点数比复数简单。混合类型自动升级。可以手动调用内置函数进行强制类型转换。
  • 位操作x << y, x >> yX左移右移Y位,X|Y位或，X^Y位异或，X&Y位与。将整数化为二进制进行操作。
  • 内置math模块

sum函数作用于数字的一个序列，min和max接受一个参数序列或者单个参数。

  • round()函数，0.5及以下去掉小数，大于0.5进1。round(x,n)x保留n位小数。
  • random()模块，需要import random导入

小数对象

小数通过一个导入的模块调用函数后创建的，而不是通过运行常量表达式创建的。小数有固定的精度的浮点值。
调用decimal模块中的Decimal的构造函数创建一个小数对象，并传入一个字符串，字符串有我们希望在结果中显示的小数位数。当不同精度的小数在表达式中混编时，python自动升级为小数位数最多的。
设置全局精度decimal.getcontext().prec=4
