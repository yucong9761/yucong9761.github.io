列表

列表是一个任意类型的对象的位置相关的有序集合，没有固定的大小。其大小是可变的，通过对偏移量进行赋值以及其他各种列表的方法进行调用，确实能够修改列表的大小。
列表list：存储不需要事先声明，可以不按照顺序存入数据，同属于一个列表的数据可以是不同类型的。存储在同一个列表中的数据都是以数字来作为索引的。使用列表时，只要在存储的多个数据两侧加中括号就可以。列表从0开始。
列表中还可以存放另一个列表，取值时需要再多一个中括号索引值

输出10

序列的操作

列表是序列的一种，所以，可以用上一章字符串里序列的操作对列表进行操作。

特定的操作

列表：（假设lst是列表变量，n、n1、n2代表某一数值，s是字符串变量）
lst*n：把列表lst重复n次
lst[n1:n2]：索引n1到n2的列表内容取出，组成另一个列表
lst[n1:n2:k]：同上，但间隔为k
del lst[n1:n2]：删除索引值n1到n2之间的元素
del lst[n1:n2:k]：同上，但间隔为k
lst[n1:n2]=n ：把索引值n1到n2之间的元素设为n
lst[n1:n2:k]=n：同上，间隔为k
len(lst)：返回列表的个数
min(lst)：返回列表中最小值
max(lst)：返回列表中最大值
lst.index(n)：返回第一次出现n的索引值
lst.count(n)：计算n在列表中出现的次数
x表示列表元素或是另一个列表变量
lst.append(x)：将x视为一个元素，附加到列表后面
lst.extend(x)：将x所有元素附加到列表后面
lst.insert(n,x)：将x插入索引值为n的地方
lst.pop()：弹出列表最后一个，可加参数指定索引
lst.remove(x)：从列表中删除第一个出现的x
lst.reverse()：反转列表顺序
lst.sort()：将列表元素内容加以排序

边界检查

列表没有固定的大小，但是超过列表末尾之外的索引或错误会导致错误。L[99]=1python中不会默默增大列表，会进行报错。为了让列表增大，可以调用append这样的列表方法。
