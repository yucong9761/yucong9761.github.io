元组

元组是序列，不可改变，编写在圆括号中，支持任意类型、任意嵌套以及常见的序列操作。

eg：

tuple.index(n)：返回第一次出现n的索引值
tuple.count(n)：计算n在列表中出现的次数
元组一旦创建，不可改变eg:T[0]=2错误。元组是不可变的序列。支持混合类型和嵌套，但不能增长缩短eg:T=('spam',3.0,[11,22,33])

元组好处

元组的不可变性提供了一种完整性约束

文件

要创建一个文件对象，需要调用内置的open函数，以字符串的形式传递给它一个外部的文件名以及一个处理模式的字符串。注：对于脚本而言，文件内容总是字符串，无论文件包含的数据是什么类型。
