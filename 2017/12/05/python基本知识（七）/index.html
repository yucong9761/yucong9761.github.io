字符串

字符串：有序的字符的集合。无单个字符，只有一个字符的字符串。字符串中字符存在从左到右的位置顺序。

字符串常量

  • python字符串中，单引号和双引号可以互换。
  • python自动在任意的表达式中合并相邻的字符串常量。
  • 在字符串之间增加逗号会创建一个元组。
  • 转义字符：\a响铃、\b倒退、\f换页、\r返回
    raw字符串抑制转移转义：字母r（大小写均可）出现在字符串第一个引号前面，则关闭转义机制，即\作为常量来保持
  • 三重引号编写多行字符串块。python将三重引号之内的文本收集到一个单独的多行字符串中，并在代码折行处嵌入’\n’。输入是什么，得到的就是什么。

实际应用中的字符串

  • 基本操作
    字符串通过’+’进行合并，’*’进行重复，len()返回一个字符串长度，in对字符和子字符串进行成员关系测试。

  • 索引和分片
    字符串偏移量从0开始，并比字符串长度小1，支持负偏移量。分片s[开始:结束:步长]结束项不包括，如果开始没有，默认为0，结束没有，默认最后一个（包括）。

  • 字符串转换
    int()将字符串转换为数字，float()相同。str()将数字转换为字符串。repr()将一个对象转换为其字符串形式。
    单个的字符ord()转换为其对应的ASCII码，chr()将ASCII码转化为对应的字符。
  • 修改字符串
    修改字符串通过合并、分片来建立并赋值给一个新的字符串。S.replace(c1:c2)将S中的c1（任意长度）变为c2（任意长度）。

    字符串方法实例

  • 修改
    find()函数，find(c1)返回在子字符串出现处的偏移（默认从前向后搜索）或未找到时返回-1。
    replace()函数，replace(c1，c2，num)把前num个c1替换为c2。
    join()函数，将列表合成一个字符串。

  • 文本解析
    分析结构并提取子串。
    split()函数，将一个字符串以分隔符字符串为标准分割为一个子字符串的列表，没有传递分隔符默认为空格。
    endwith(c1),startwith(c2)以c1结尾，以c2开头。

字符串格式化

  • 格式化字符串：
    1.在%操作符的左侧放置一个需要进行格式化的字符串，这个字符串带有一个或多个嵌入的转换目标，都以%开头（如：%d）
    2.在%操作符右侧放置一个（或多个，嵌入到元组中）对象，这些对象将会插入到左侧想让python进行格式化字符串的一个（或多个）转换目标的位置上去。
    "That is %d %s bird" % (1,"dead")
    格式化总会返回新的字符串作为结果而不是对左侧的字符串进行修改
    左对齐（-），正负号（+），补零（0），可以在格式化字符串中用一个*来指定通过计算得出的width和precision，它们的值从%运算符右边的输出中的下一项获取。
    "%f,%.2f,%.*f"%(1/3.0,1/3.0,4,1/3.0)

  • 基于字典的字符串格式化
    左边的转换目标来引用右边字典中的键来提取对应的值。"%(n)d %(x)s" % {"n":1,"x":spam}

  • 字符串格式化调用方法
    format()方法使用主体字符串作为模板。在主体字符串中，花括号通过位置（{1}）或关键字（{food}）指出替换目标及将要插入的参数。

format创建并返回一个新的字符串对象，可以立即打印或保存起来。
格式化字符串中的方括号可以指定列表（及其他序列）偏移量以执行索引，只有单个的正的偏移才能在格式化字符串的语法中有效，要指定负的偏移量或分片或使用任意表达式必须在格式化字符串自身之外运行表达式
