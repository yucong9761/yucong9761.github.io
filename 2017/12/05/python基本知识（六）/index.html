动态类型简介

python中，类型在运行过程中自动决定，不是通过代码声明。

变量在赋值时才创建，可以引用任何对象，并在引用之前赋值。eg：a=3创建一个对象代表值3，创建一个变量a，将变量与新的对象3相连接。类型属于对象，不是变量名。

回收

python在每个对象中保持了一个计数器，计数器记录了当前指向该对象的引用的数目。一旦这个计数器被设置为0，这个对象的内存空间自动回收。

共享引用

多个变量名引用同一个对象。

共享引用与相等

“==”两个被引用对象是否有相同的值，”is”检查对象的同一性，只针对列表等较复杂情况，小的数字例如int型字符串型的被缓存并复用了。
